# MVP Roadmap

## Phase 1 - Foundation
- Nx workspace + Ionic app.
- Tourism feature slice scaffold.
- Playwright baseline smoke tests.

## Phase 2 - Map + POIs
- Google map render + marker cluster.
- POI list/summary + detail drawer.
- Viewport/city query filters.

## Phase 3 - Routing
- A->B route calculation + ETA.
- Route alternatives and transport modes.
- Error and slow-network states.

## Phase 4 - Hardening
- BFF caching and rate-limit protection.
- Offline cache and stale UI strategy.
- E2E regression matrix + telemetry.
